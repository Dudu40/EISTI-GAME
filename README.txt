EISTI Game "Battle"

-----------------------------

Assurez vous d'importer projetbdd.sql dans votre BDD de choix avant tout.
Dans ./scripts/fonctionsbdd.php, changez les variables $USER, $PASS, et $BASE avec vos propres informations. (Respectivement, le pseudo, le mot de passe, et le nom de la BDD.)

Vous pouvez maintenant accéder à connexion.php, un compte est déjà crée auparavant, nommé "test", ayant comme mot de passe "test" aussi, et possédant les droits d'administrateur.
Vous avez maintenant accès à toutes les fonctionalités du logiciel, amusez vous bien!

(PS : Notre meilleur score est à 15 000.)
